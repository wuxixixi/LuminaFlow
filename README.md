# LuminaFlow

**版本**: 1.4.2

LuminaFlow 是一款基于 AI 的图片转视频工具，使用 DMXAPI（MiniMax Hailuo 2.3）API 将静态图片转换为动态视频。支持批量处理，提供直观的图形界面和命令行两种使用方式。

## 功能特点

- **批量转换**: 支持批量选择多张图片进行视频转换
- **实时预览**: 列表显示图片缩略图、尺寸和转换状态
- **智能检测**: 自动检测已存在的视频文件，避免重复转换
- **多种动效**: 内置多种提示词模板，支持自定义动态效果
- **并发处理**: 支持最多 4 个并发任务，提高转换效率
- **进度追踪**: 实时显示转换进度和状态

## 系统要求

- Windows 7 或更高版本
- 网络连接（用于 API 调用）

## 快速开始

### 1. 获取 API Key

1. 访问 [DMXAPI 平台](https://www.dmxapi.cn)
2. 注册并获取 API Key

### 2. 配置 API Key

有两种方式配置 API Key：

**方式一：通过界面设置**
1. 启动程序后点击「设置」按钮
2. 在 API Key 输入框中粘贴您的 Key
3. 点击「保存」

**方式二：通过环境变量**
1. 创建 `.env` 文件
2. 添加内容：`DMXAPI_API_KEY=your-api-key-here`

### 3. 添加图片

支持以下方式添加图片：

- **选择文件夹**: 点击「选择文件夹」批量导入整个文件夹的图片
- **添加图片**: 点击「添加图片」选择单个或多个图片文件
- **拖拽**: 直接将图片文件拖拽到程序窗口

支持的图片格式：JPG、PNG、WEBP

### 4. 开始转换

1. 在列表中勾选需要转换的图片（默认全选）
2. 点击「开始转换」
3. 等待转换完成

### 5. 查看结果

转换完成后，点击列表项右侧的「打开视频」按钮可直接播放视频。视频文件保存在 output 目录中。

## 图片要求

为确保最佳转换效果，图片应满足以下条件：

- **最小尺寸**: 短边不小于 300 像素
- **宽高比**: 0.4 ~ 2.5 之间
- **文件大小**: 不超过 20MB
- **格式**: JPG、PNG、WEBP

## 常见问题

### Q: 转换失败怎么办？

A: 请检查以下事项：
1. API Key 是否正确配置
2. 网络连接是否正常
3. 图片是否符合要求
4. 账户余额是否充足

### Q: 转换需要多长时间？

A: 每张图片的转换时间约为 1-3 分钟，具体取决于服务器负载和视频时长设置。

### Q: 可以取消正在进行的转换吗？

A: 可以。点击「停止」按钮可以取消当前的批量转换任务。

### Q: 视频保存在哪里？

A: 默认保存在程序目录下的 `output` 文件夹中，可以在设置中修改输出目录。

### Q: 如何重新转换已完成的图片？

A: 点击列表项右侧的「再次转换」按钮，或者右键菜单中选择相应选项。

## API 模式

LuminaFlow 支持 API 服务模式，允许外部程序调用进行图片转视频。

### 启动 API 服务

```bash
LuminaFlow.exe -api              # 默认端口 8080
LuminaFlow.exe -api -port 9000   # 指定端口
```

### API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/info` | 服务器信息 |
| POST | `/api/convert` | 转换单张图片 (JSON) |
| POST | `/api/upload` | 上传并转换图片 (multipart) |
| POST | `/api/batch` | 批量转换 |
| GET | `/api/status/:id` | 查询任务状态 |
| GET | `/api/tasks` | 获取所有任务 |
| GET | `/api/download/:filename` | 下载视频 |
| POST | `/api/stop/:id` | 停止任务 |
| POST | `/api/stop` | 停止所有任务 |

### 使用示例

**转换单张图片：**
```bash
curl -X POST http://localhost:8080/api/convert \
  -H "Content-Type: application/json" \
  -d '{"image_path": "C:/images/photo.jpg"}'
```

**上传图片转换：**
```bash
curl -X POST http://localhost:8080/api/upload \
  -F "image=@photo.jpg"
```

**查询任务状态：**
```bash
curl http://localhost:8080/api/status/任务ID
```

## 技术架构

- **开发语言**: Go
- **GUI 框架**: Fyne
- **API**: DMXAPI (MiniMax Hailuo 2.3)

### API 工作流程

1. **编码**: 将图片转换为 Base64 格式
2. **提交**: 调用 API 提交视频生成任务
3. **轮询**: 定期查询任务状态，直到完成
4. **下载**: 获取视频文件并保存到本地
